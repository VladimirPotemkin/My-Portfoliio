# spa-resume

Текущий статус: завершены `Этап 1`, `Этап 2`, `Этап 3`, `Этап 4`, `Этап 5`, `Этап 6`, `Этап 7`, `Этап 8` и `Этап 9`.

Коротко:

- проект работает на `bun`;
- базовый Vue SPA bootstrap уже собран;
- доменные типы и профиль вынесены в отдельный слой данных;
- визуальная система переведена на CSS-токены и семантические классы;
- подготовлены светлая и тёмная темы через класс `dark` на `html`.
- собран общий SPA-layout с шапкой, футером, маршрутами и переходами между страницами.
- глобальные настройки и GitHub-данные централизованы через Pinia stores.
- навигация переведена в формат one-page scroll: шапка плавно прокручивает к секциям вместо переключения отдельных страниц.
- подключён `vue-i18n`: интерфейс уже переключается между `RU` и `EN`, а выбранная локаль сохраняется между сессиями.
- главная секция оформлена как полноценный portfolio-landing с таймлайном опыта и модальным CV.
- портфолио работает на реальных данных GitHub: есть загрузка, ошибки с retry, фильтры по языкам и карточки проектов с превью.
- в контактах реализованы копирование email через `useClipboard`, глобальные toast-уведомления и форма обратной связи с валидацией.
